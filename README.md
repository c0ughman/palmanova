# Palmanova Center — Case Study

Standalone case study page for **Palmanova Center** by Jasper Architects.

Deployed at [palmanova.netlify.app](https://palmanova.netlify.app).

## Structure

- `index.html` — full case study page
- `hero/` — hero slideshow images and method videos
- `imgs/projects/00038/` — project photography
- `css/fonts/` — Jasper Architects brand fonts

## Local preview

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Deploy

Static site — publish root directory to Netlify. No build step required.
